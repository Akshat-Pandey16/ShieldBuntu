# shieldbuntu-web

The React SPA for ShieldBuntu. Talks to the FastAPI backend at `/api/*` (proxied to `localhost:8000` in dev).

## Stack

- **React 19** + **TypeScript 6** (strict)
- **Vite 8** dev server / bundler
- **Tailwind v4** via `@tailwindcss/vite` (CSS-based `@theme` config — no `tailwind.config.js`)
- **TanStack Router** (file-based routing in `src/routes/`)
- **TanStack Query** for server state
- **shadcn/ui** components added via the CLI on demand
- **Vitest** + **Playwright** for tests

## Scripts

```bash
pnpm dev          # dev server on :5173 with HMR
pnpm build        # typecheck + production build
pnpm preview      # serve built bundle
pnpm typecheck    # tsc -b --noEmit
pnpm lint         # eslint
pnpm format       # prettier --write
pnpm test         # vitest watch
pnpm test:e2e     # playwright
pnpm gen:api      # regenerate typed API client from backend OpenAPI
```

## Notes

- `src/routeTree.gen.ts` is generated by `@tanstack/router-plugin` on Vite startup. Don't edit.
- `src/lib/api.gen.ts` will be generated from FastAPI's `/openapi.json` once the API stabilizes (Phase 2).
- Tailwind v4 theme tokens live in [src/styles.css](src/styles.css); add CSS vars under `@theme`.
